Welcome to the cURL challenge!

-- --=[Challenges]=-- --

There is no order to the challenges for this lab so have a go at any ones you want:

Challenge 1 - To get the encoding challenge for this one send a GET request with two parameters, one called whoAreYou with the value AHacker and the other called areYouSure with the value yes


Challenge 2 - To get this encoding challenge, make your request use HTTP/1.0


Challenge 3 - HTTP 1.1 has a fair few request methods that are documented. In this case make a HTTP PATCH request for your encrypted challenge


Challenge 4 - POST some JSON content for this one (in a similar fashion to challenge 1), in your POST requests JSON body, set "WeGonnaRockDownTo" to "ElectricBoogaloo". This challenges errors are a bit more helpful, you might want to look at a tool called jq also.


Challenge 5 - An easier cURL challenge now that we're near the end. Send a GET request to port 8080 for this one


Challenge 6 - Start by running `echo "superMegaFinalFlagFile" > file.txt` then upload that file while using the HTTP PUT method


Notes

The challenges are made up of two parts: Using cURL to send the correct response and decoding the challenge string to get the flag


Feel free to use wget for style points or work out how to use any other program if you're interested


For hints append hint1, hint2 or hint3 to the challenge url e.g. `curl http://localhost/challenge3/hint1`. (Not all challenges have the same number of hints)
